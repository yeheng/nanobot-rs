Welcome to Nanobot

The web client has been completely rewritten into a feature-rich Vue 3 application! 🎉

New features include:

  • History Logging: Seamless saving and restoration of past conversations.
  • Multi-Window & Multi-User: Create discrete sessions that run in parallel.
  • Rich Interaction: Markdown support, elegant collapsible Thinking UI, and structured Tool Call responses.

How to start the new client:

cd web && npm install && npm run dev
